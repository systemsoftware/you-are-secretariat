# You Are Secretariat

This is a simple desktop application built with `positron.js` that uses your webcam to recreate the iconic "You Are Secretariat" mirror billboard from *Bojack Horseman*. It opens a frameless window showing your webcam feed with the "YOU ARE SECRETARIAT" text overlay.

## About the Reference
In Season 2 of *Bojack Horseman*, the marketing team for BoJack's dream movie, *Secretariat*, launches a promotional campaign consisting of reflective billboards with the text "YOU ARE SECRETARIAT". The idea is that anyone looking at the billboard sees their own reflection, making them feel like they embody the spirit of the famous racehorse. This app brings that promotional mirror to your desktop using your webcam.

## Features
- Live webcam feed to act as a "mirror".
- Overlays the text "YOU ARE" and "SECRETARIAT" in a varsity font.
- Frameless, hidden-titlebar window setup for a clean mirror look.
- Right-click context menu to toggle window traffic lights (on macOS).

## Prerequisites
- Node.js installed on your machine.
- A working webcam.

## Installation

1. Clone or download the repository.
2. Install the dependencies:
   ```bash
   npm install
   ```

## Usage

Run the application using Node:
```bash
node index.js
```
